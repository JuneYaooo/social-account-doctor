# 视频号账号 / 内容诊断子手册

> 主 skill：`account-diagnostic`。本手册被 §3 平台专属诊断章节调用。
> **核心特性**：视频号是**社交关系链分发**平台 — 朋友点赞 / 转发 / 在看会推送给微信好友，**冷启动靠分享而不是算法推流**。算法逻辑跟抖音、小红书完全不同。

---

## 0. 数据来源声明

视频号官方公开的算法数据极少，本手册阈值均为**行业经验值 + 创作者社区共识**：

- 微信公开课（2023 / 2024 / 2025）— 关于视频号社交分发机制的官方表述（"朋友推荐 + 机器推荐 + 关注流"三段式）
- 新榜 / 西瓜数据 / 友望数据 — 视频号头部账号公开数据反推
- 微信生态创作者社群（KK 黄有璨 / 朋友圈刷屏案例集） — 分享率 / 朋友圈点赞率经验值

**API 数据可得性**：tikhub-wechat MCP **完整覆盖**视频号（视频详情 / 下载 / 搜索 / 用户 / 评论 / 热榜）+ 公众号（文章 / 阅读量 / 评论），见 §4。但视频号**没有像抖音星图 / 小红书蒲公英那样的开放创作者后台 API** — 私有指标（分享率 / 朋友点赞率 / 关注转化）只能从用户提供的视频号助手 App 截图里拿。所以实战中**用户输入主要是「链接 + 截图 + 描述」混合**，工具链按 §4.1 路由。

⚠️ **视频号算法不透明度比抖音 / 小红书更高**，本手册数字仅作方向判断；**核心定性结论"分享 > 一切"在多源反复验证，是稳的**。

📅 **数据采集日期：2026-04 / 建议复核周期：6 个月**。视频号 2024-2026 经历了"朋友推荐占比下调 / 机器推荐占比上调"的算法调整，但社交分享仍是冷启动唯一通道。

## 0.5 算法规则速查 — T0/T1（可执行版）

> ⭐ **算法核心信号 = 分享率**（朋友圈 / 私聊 / 群聊 / 在看 / 转发）。差距 ≥ 1 档时**优先于其他维度修** — 视频号没有抖音那种"陌生流量大池子"，**冷启动 90% 靠社交关系链**：朋友点赞 → 推送给所有共同好友。没有分享触发点 = 算法连第一个推荐池都不给。
>
> T0 = 不做就死；T1 = 不做次推。

| 级 | 规则 | 怎么做（动作） | 不做的代价 |
|---|---|---|---|
| ⭐ **T0** | 每条视频必须埋 1 个"分享触发点" | 金句 / 奇观 / 共鸣点 / 实用信息 — 让人想转给朋友 / 群 / 朋友圈 | 没有分享 = 没有社交关系链分发 = 冷启动死 |
| ⭐ **T0** | 内容要"朋友圈友好"（无社交压力） | 朋友点赞会推给所有共同好友 — 内容不能让朋友"不好意思被人看到点过赞"（避硬广 / 三观争议 / 低俗） | 朋友看到广告会取消点赞，分享链断裂 |
| ⭐ **T0** | 视频时长 60s-3min 是甜区 | < 30s 信息密度不足以触发分享；> 5min 朋友圈缩略图体验差 | 分享率掉一半 |
| **T0** | 标题 = "对话感 + 共鸣 / 价值"，禁止抖音腔 | 「这条视频我刷到三次还想看」/「家人们必看」（社交语气）— 不要「3 个技巧让你 XXX」（抖音知识腔） | 不像朋友推荐，朋友圈点赞率掉 |
| **T1** | 封面 = 16:9 横版或 1:1 方版（朋友圈缩略图友好） | 不要 9:16 抖音竖版（朋友圈会被裁掉关键信息） | 朋友圈点击率掉 30%+ |
| **T1** | 开头 3-5s 必有钩子（不像抖音那么严苛但仍要） | 朋友点开后 3s 内还是要抓住，但允许"叙事开场"（不必倒叙） | 完播率低 → 算法判定不值得继续推 |
| **T1** | 主页关联公众号 + 微信号（私域承接） | 发布时勾选"关联公众号"+ 简介留微信引导 | 流量来了接不住 = 没有变现路径 |

## 1. 阈值表（行业经验值，非官方）

| 指标 | 计算 | 偏低 | 合格 | 优秀 | 卡住意味着 |
|---|---|---|---|---|---|
| **分享率**（命门） | 转发 / 播放 | < 0.5% | 0.5-2% | > 2% | 没有"愿意推给朋友"的钩子 |
| **朋友点赞率** | 朋友圈来源点赞 / 总点赞 | < 20% | 20-50% | > 50% | 内容没击穿熟人圈层 |
| 完播率 | 完整看完 / 点击 | < 25% | 25-45% | > 45% | 节奏断 / 中段塌 |
| 互动率 | (赞+评+转) / 曝光 | < 1% | 1-3% | > 3% | 选题没共鸣 |
| 关注转化率 | 新关注 / 曝光 | < 0.1% | 0.1-0.5% | > 0.5% | 人设模糊 |
| **公众号关联跳转率** | 跳公众号 / 视频曝光 | < 0.5% | 0.5-2% | > 2% | 没用上视频号→公众号→私域的微信生态优势 |

**经验铁律**：视频号看分享率比看播放量重要 10 倍。一条 1 万播放分享率 3% 的视频，比一条 10 万播放分享率 0.3% 的更值钱（前者会持续滚雪球，后者就是流量泡沫）。

## 2. 主逻辑 1：分享率诊断（视频号头号信号）

### 2.1 怎么定位是不是卡在这一层

```
1. 漏斗看到 分享率 < 0.5% 或 朋友点赞率 < 20% → 直接进本节
2. 调 mcp__tikhub-wechat__* 拿视频详情 + 评论
   → 看转发数 / 在看数 / 朋友圈点赞数
3. 看视频本身：是否有"金句 / 奇观 / 共鸣 / 实用信息"4 类分享触发点中的至少 1 个
4. 对照 §2.2 五种分享失败模式定位根因
```

### 2.2 分享率崩盘的五种典型根因

| 根因 | 现象 | 修法 |
|---|---|---|
| **没分享触发点** | 内容平铺直叙，没有"必须让朋友看到"的瞬间 | 全片至少 1 个金句 / 奇观 / 强共鸣，剪辑时把它放在 30s-90s 高潮位 |
| **社交压力** | 内容是硬广 / 鸡汤 / 争议 — 朋友不想被看到点过赞 | 改成中性价值（实用 / 治愈 / 美 / 知识），避开"立场表态" |
| **朋友圈缩略图差** | 9:16 竖版被裁掉关键信息 / 封面无字 | 改 16:9 或 1:1 + 大字标题（朋友圈缩略图也要能"读懂"） |
| **时长错位** | < 30s 没立稳价值 / > 5min 朋友圈缩略图劝退 | 60s-3min 甜区 |
| **标题抖音腔** | "3 个技巧让你 XXX"这种知识带货腔 — 朋友圈不会推 | 改对话感："这条我刷了三次"/"家人们必看" |

### 2.3 4 种分享触发点（必有其一）

| 类型 | 描述 | 例子 |
|---|---|---|
| **金句型** | 一句话能截屏发朋友圈的 | 「成年人的崩溃都是静音的」 |
| **奇观型** | 没见过的画面 / 反常识 | 「猫咪 24 小时拟人 vlog」 |
| **共鸣型** | 戳中特定人群的集体记忆 | 「90 后小时候的零食」 |
| **实用型** | 收藏 + 转发的工具 / 清单 | 「30 个 Excel 神技」 |

## 3. 主逻辑 2：社交关系链分发诊断

### 3.1 视频号三段式分发机制（微信公开课口径）

```
阶段 1：关注页（500-1000 粉丝池）
  └ 你发布 → 推给关注你的人 → 看完成率决定下一步

阶段 2：朋友推荐（社交关系链 — 视频号最大特色）
  └ 朋友点赞 / 在看 / 转发 → 推送给所有共同好友
  └ "朋友 ❤️" / "X 个朋友推荐" 是这个池子

阶段 3：机器推荐（陌生人池子）
  └ 阶段 1+2 表现好 → 进通用推荐池
  └ 但仍受社交信号加权，不是纯算法
```

### 3.2 阶段过不去的根因

| 卡哪 | 现象 | 修法 |
|---|---|---|
| **过不了阶段 1** | 关注页发完 24h 播放 < 粉丝数 ×0.5 | 看是不是发布时段错了 / 标题没钩 / 封面崩 |
| **过不了阶段 2** | 阶段 1 OK 但朋友点赞率 < 20% | 内容没"共同好友会感兴趣"的勾子 — 改话题或共鸣点 |
| **过不了阶段 3** | 朋友圈刷屏了但陌生人池没起来 | 完播 / 互动率不够 — 节奏 / 中段优化 |

### 3.3 微信生态联动（视频号专属优势）

视频号 + 公众号 + 微信群 + 朋友圈 + 小程序 = **五合一闭环**。其他平台没有的红利：

- 视频号挂载公众号文章 → 引流到深度内容
- 视频号嵌入微信群 → 群内播放数算分享
- 视频号 → 微信小商店 → 直接交易
- 朋友圈分享卡片 → 触达所有微信好友

**P0 必做**：发布时关联公众号 + 简介引导加微信。

### 4.1 输入路由（视频号 ≠ 抖音/小红书）

视频号**没有像抖音星图 / 小红书蒲公英那样的开放创作者后台 API** — 个人创作者只能从视频号助手 App 截图拿数据。所以实战中用户给的输入按下面优先级路由：

```
1. 视频号账号名 / 关键词       → 走 §4.2 锁定本号 → 拿 video id → tikhub-wechat MCP
2. 用户给的后台截图           → ocr_screenshot.py 提分享率 / 朋友点赞率 / 关注转化等私有指标
3. 用户口述描述               → 直接做诊断，标注"未实测，基于描述推断"
```

⚠️ **不要等用户提供"视频号链接"** — 视频号客户端**只支持卡片分享，根本不输出 URL/视频 ID**。任何视频号任务唯一入口都是账号名/关键词搜索。

⭐ **铁律**：诊断视频号必须**至少拿到分享率**这一个数（命门指标）。如果用户给的截图没有，必须明确问："分享数 / 转发数是多少？" 不要凭点赞/播放反推 — 视频号的算法权重跟其他平台完全不同。

### 4.2 user_search 503 兜底流程（**实战必背**）

`user_search` 是 tikhub-wechat 里**最不稳定**的接口（实测 2026-04-21 高频 503）。锁定本号永远先走 `_search_ordinary`：

```
输入：账号名（如「哈吉老猫」）

Step 1  wechat_channels_fetch_search_ordinary(keywords=账号名)
        → 拿到 items[]，每条含 source.title / source.iconUrl

Step 2  本号判定（必须双重校验，缺一不可）：
        a. 剥掉 source.title 里的 <em class="highlight"> 高亮 HTML
        b. 剥后字符串与目标账号名 **完全相等**（不是包含）
        c. iconUrl 在多条本号视频间一致（同一个 finderhead URL）
        → 满足 a+b+c 的才是真号

Step 3  从该条拿 exportId / hashDocID
        → wechat_channels_fetch_video_detail(exportId 或 id)
        → 返回的 contact 节点里就有 nickname / username / feed_count / signature / live_status / ip_region

Step 4  跳过 user_search，直接进诊断
```

**为什么不能只看 source.title 包含目标名**：搜索"哈吉老猫"会同时出"是哈吉仙人"、"哈基米"、"老猫成仙"等同主题号，包含关键字 ≠ 是本号。**真号常常只有 1 条**（综合搜索 12 条结果里只 1 条匹配）。

### 4.3 冷启失败号秒判公式

`video_detail` 返回里有两个杀手字段，组合使用可在**单次 API 调用**判定账号是否冷启失败：

| 字段 | 来自 | 冷启失败信号 |
|---|---|---|
| `data.contact.feed_count` | video_detail | **= 1**（只发了 1 条 = 没有内容矩阵） |
| `data.like_count` / `comment_count` / `forward_count` / `fav_count` | video_detail | **全 = 0**（21 天 0 互动 = 没进推荐池） |
| `data.contact.signature` | video_detail | **= ""**（简介空 = 算法/用户都不知你是谁） |
| `data.contact.live_status` | video_detail | `2` = 可开播但近期无直播 |

**判定规则**：上述 4 项至少**命中 3 项** = 冷启失败号 → 直接出 P0 诊断，不需要再拉别的接口。

### 4.4 赛道错位诊断（账号名 vs 内容预期）

视频号有大量"借势热梗"的账号名（哈基米 / 哈吉 / 耄耋 / 老猫等），用户搜该账号名时**预期是猫梗短视频**。如果账号实际内容与名字暗示的预期错配，**会被算法 + 用户双重抛弃**：

```
诊断流程：
1. 用 _search_ordinary(账号名) 看综合搜索 top 12 结果的内容类型分布
   → 大多数是猫梗 / 耄耋 / AI 萌宠 → 这就是该名字的"赛道预期"
2. 用 video_detail 看本号实际内容的 description + 标签
3. 对比：内容主题 vs 赛道预期
   - 命中 → 正常
   - 完全错配（如名字暗示猫梗 / 实际发 AI 翻唱）→ P0 警告：要么改名，要么改内容
```

**实战案例**（2026-04 实测「哈吉老猫」）：账号名暗示猫梗赛道，搜索结果 12 条里 11 条是耄耋/哈基米/AI 胖猫；本号唯一作品是 AI 翻唱歌曲 → 完全错配 → 21 天 0 互动 → 冷启失败。修法：要么去"哈吉"前缀改成 AI 音乐人设，要么改拍真猫 + 哈基米梗。

## 5. 视频号版六维评分细则

| 维度 | 视频号特化指引 |
|---|---|

## 7. 关键铁律（视频号专属）

1. **分享 > 一切**。分享率比播放量、点赞、关注都重要 10 倍。
2. **没有"陌生流量大池子"**。冷启动 90% 靠社交关系链 — 朋友点赞推给共同好友。
3. **朋友圈友好原则**：朋友点赞会被全部好友看到，所以内容不能让朋友"不好意思"。
4. **不要套抖音逻辑**。9:16 竖版 / "3 招 5 招"知识腔 / 高密度快剪 在视频号会扑街。
5. **微信生态联动是独家红利**。视频号 → 公众号 → 微信群 → 朋友圈 → 小程序，五合一闭环没人能抄。
6. **直播是终极变现**。视频号直播带货增速最快，但需要先在短视频积累信任。
7. **算法不透明，定性结论稳，定量数字疑**。本手册数字仅供方向判断，每 6 个月复核。
