# 快手账号 / 内容诊断子手册

> **平台调性**：老铁经济 + 同城下沉 + 强信任关系。算法不靠"广撒网爆款"，靠"小池子高粘性"。
> 来本手册前先把 SKILL.md §1 三层诊断框架记住 — 这里只补**快手专属经验阈值 + 工具 + 评分细则**。

## 0. 数据来源声明

本手册所有阈值均为**行业经验值**（无快手官方公布的诊断阈值表），主要来源：

- 券商研究报告 — [东方财富《抖音、快手生态成因》](http://pdf.dfcfw.com/pdf/H3_AP202010201422479434_1.pdf) / [《快手商业化的进化与突围》](https://pdf.dfcfw.com/pdf/H3_AP202101211452765930_1.pdf) / [《快手：由慢向快》](https://wallstreetcn.com/articles/3607502)（多份均提到"同城/关注流量占比远高于抖音"，但**未给精确比例**）
- 知乎运营拆解 — [《快手短视频流量及推荐算法解析》](https://zhuanlan.zhihu.com/p/362863254) / [《不同短视频平台流量本质区别》](https://zhuanlan.zhihu.com/p/1911060707308450483)
- 经验数字 — [亿专 5《快手五秒完播率》](https://www.yizhuan5.com/yarticle/4552.html) / [搜狐《快手养号注意事项》](https://www.sohu.com/a/886622464_122317756)

⚠️ **本手册中"同城占比 30-50%""关注转化率 1.5%""评论率 1.5%"等具体数字均无可靠官方出处**，是创作者社区流传的经验值，仅作方向判断；**核心定性结论"快手关系沉淀 > 内容分发"反复多源验证，是稳的**。

📅 **数据采集日期：2026-04 / 建议复核周期：6 个月**。快手算法相对稳定（大盘逻辑多年不变），但具体阈值随商业化进展会调，建议半年复核一次。**定性结论（关系沉淀、同城权重高、人设>内容）多年不变，可长期信任**。

## 0.1 平台一句话定位

快手 ≠ 抖音的下沉版。
**抖音是内容分发平台**（看完就走），**快手是关系沉淀平台**（看完会关注、会评论、会私信、会复购）。
所以诊断快手别看"单条爆没爆"，看 **"这条带来多少新关注 + 评论区互动质量有没有起来"**。

## 0.5 算法规则速查 — T0/T1（可执行版）

> ⭐ **算法核心信号 = 跳出 / 留存 + 关注转化**。差距 ≥ 1 档时**优先于其他维度修** — 快手不靠"广撒网爆款"，靠"小池子高粘性"。被划走 = 老铁信任建立失败；不关注 = 没有沉淀价值。
>
> T0 = 不做就死；T1 = 不做次推。

| 级 | 规则 | 怎么做（动作） | 不做的代价 |
|---|---|---|---|
| ⭐ **T0** | 5s 跳失率 ≤ 50%（即 5s 完播 ≥ 50%） | 首句"老铁们好 + 痛点钩子"，禁止 5s 内静止 / 黑屏 / 自我介绍 | 系统判定老铁不喜欢，不进同城 / 关注扩量 |
| ⭐ **T0** | 关注转化率 ≥ 1%（核心命门） | 必出镜 + 开头报名号 + 结尾"关注 XX 看 YY" + 立固定人设 | 流量起来也没沉淀 = 几条号就废 |
| **T0** | 每条视频必打位置标签（定位） | 发布时勾选当前定位，打到区 / 街道级 | 主动放弃 ~30% 同城流量入口 |
| **T0** | 评论 24h 内必回 + 回复带 @ 粉丝名 | 「@张哥 你说得对」格式；每条评论都回 | 评论区死气，算法不给社区流量 |
| **T1** | 固定 3 个栏目（不要散） | 选 3 个栏目反复做，禁止今天美食明天数码 | 老铁记不住你 |
| **T1** | 标题 / 口播带城市名或方言 | 「成都打工人 / 义乌商家 / 东北老铁」 | 同城算法识别不到 |
| **T1** | 直播每周 ≥ 2 次，每次 ≥ 1 小时 | 培养老铁打赏 / 私信 | 没直播 = 没信任沉淀，复购为 0 |

## 1. 漏斗阈值表（行业经验值，非官方）

| 指标 | 健康阈值 | 警戒线 | 偏低 | 卡住意味着 |
|---|---|---|---|---|
| 曝光率（曝光/粉丝） | ≥ 3.0 | 1.5 - 3.0 | < 1.5 | 账号权重低 / 标签错乱 |
| CTR（点击/曝光） | ≥ 8% | 5 - 8% | < 5% | 封面 + 标题不及格 |
| 5s完播率 | ≥ 50% | 35 - 50% | < 35% | 钩子 + 节奏问题 |
| 整体完播率 | ≥ 30% | 20 - 30% | < 20% | 中段拖沓 / 没结构 |
| **关注转化率**（新关注/点击） | **≥ 1%** | 0.5 - 1% | < 0.5% | **快手核心关注指标** — 没人关注=人设崩 |
| 评论率（评论/曝光） | ≥ 1% | 0.5 - 1% | < 0.5% | 没埋互动钩子 / 不接地气 |
| 评论回复率（你回的/总评论） | ≥ 50% | 30 - 50% | < 30% | 没经营老铁关系 |

**经验铁律**：快手不看"百万赞爆款"，看 **关注转化率 + 评论质量 + 复访率**。
一条 5 万播放但带来 200 关注的视频，比一条 50 万播放只带 50 关注的更值钱（这一定性结论在多份券商研报中都有印证）。

## 2. 主逻辑 1：完播 + 关注转化率诊断（老铁经济核心）

### 2.1 完播率分层（经验值，反映"系统对内容质量的判断方向"，不是官方公布的硬门槛）

| 5s完播率 | 整体完播率 | 经验判定 | 推送动作（推测） |
|---|---|---|---|
| ≥ 50% | ≥ 30% | 优质内容 | 推大池 + 同城 + 关注页 |
| 35 - 50% | 15 - 30% | 普通 | 小池子分发后停止扩量 |
| < 35% | < 15% | 偏弱 | 小池子内不再扩量（"立刻限流停推"是社区流传，无官方背书）|

### 2.2 关注转化率诊断（快手最重要）

```
关注转化率 = 新增关注数 / 点击数（不是曝光）
```

**判定逻辑**（经验阈值，无官方出处）：
- ≥ 1%：人设清晰，"我喜欢这个人 → 关注他"
- 0.5 - 1%：内容 OK，但人设模糊
- < 0.5%：内容是孤立的"知识/搞笑"，没立人设 → 看完就走

**根因清单**（关注率低就对照看哪一条中招）：
1. **没出镜 / 没固定人设**：纯录屏 / 纯字幕 → 老铁记不住你
2. **不报名号**：开头不说"我是 XX"，结尾不说"关注 XX 看更多"
3. **内容散**：今天美食明天搞笑后天讲数码 → 关注你干嘛
4. **没"利他承诺"**：没告诉粉丝"关注我能得到什么持续价值"
5. **结尾断崖**：内容讲完直接结束，没引导关注 / 没下集预告

### 2.3 完播率提升的 3 个快手专属技巧

1. **首句报"老铁好"+ 痛点钩子**（"老铁们，今天教你一招月省 500 块"）
2. **中段必须有 1-2 次"老铁们看好了"**（提醒注意力，再钩一次）
3. **结尾 3 秒留悬念**（"下集教你 XX，关注我别错过"）

## 3. 主逻辑 2：同城分发诊断（快手独特红利）

### 3.1 同城流量占比

券商研报普遍认同"快手同城/关注流量占比远高于抖音"（[东方财富研报](http://pdf.dfcfw.com/pdf/H3_AP202010201422479434_1.pdf)），但**未给具体百分比**。社区流传的"快手 30-50% 来自同城页、抖音 10-20%"是经验估算，方向稳，具体数字别当真。

**同城分发健康度 = 同城曝光 / 总曝光**（经验阈值）：

| 同城占比 | 判定 |
|---|---|
| ≥ 30% | 健康 |
| 15 - 30% | 同城标签弱，需补 |
| < 15% | 几乎没拿到同城流量 — 严重问题 |

### 3.2 同城分发的 3 个杠杆

1. **位置标签必须打**（每条视频都加定位 — 不打 = 主动放弃 30% 流量）
2. **画面里有"地域识别物"**（招牌、方言、本地地标 → 算法和用户双识别）
3. **标题/口播带城市名 / 区域词**（"成都打工人""义乌商家""东北老铁")

### 3.3 评论质量诊断（快手 = 评论区社区）

**判定信号**（定性观察，不算公式）：
- 粉丝评论是否带名字 / @ 你？
- 是否有"复评"（粉丝来回回好几条）？
- 长评论（≥ 20 字）占比？

健康账号的评论区会出现 ≥ 30% 的"长评论 / 主动 @ / 复评"。低于这个 = 评论区死气沉沉 → 算法不会给社区流量。

**修复手段**：
- 视频结尾抛 1 个具体问题（"老铁们，你们当地也这样吗？评论区告诉我"）
- 粉丝评论必回，回复带名字（"@张哥 你说得对"）
- 每周固定 1 次"翻牌评论"（把粉丝评论拍成视频回应）

## 5. 六维评分细则（快手版）

| 维度 | 1 分（差） | 3 分（中） | 5 分（强） |
|---|---|---|---|
| **账号定位** | 没固定人设 / 内容散 | 有标签但模糊 | "XX 城市的 XX 类老铁" — 一句话能讲清 |
| **选题角度** | 跟拍流水账 | 有痛点但泛 | 同城 + 痛点 + 强人设 = 三位一体 |
| **封面公式** | 随便截 / 无字 | 有大字但乱 | 大字 + 表情 + 强对比，5m 外能读 |
| **标题钩子** | 平铺直叙 | 有疑问句 | "老铁们 + 痛点 + 反差/承诺" |
| **正文骨架** | 没结构 / 拖沓 | 有钩子但中段塌 | 首句钩子 + 中段反转 + 结尾留悬念 |
| **发布节奏** | 周更 / 时段乱 | 周 3-4 条 | 日更 + 固定时段（早 7-8 / 晚 7-9）+ 评论必回 |

**对标基准**：5k-50k 粉的快手账号平均六维得分 18-22 分（满分 30）。
你低于 15 = 全维度都得调；你 22+ = 已经是赛道头部，再调就是边际优化。

## 7. 平台专属铁律

1. **关注转化率 > 播放量**。一条 5 万带 200 关注 > 一条 50 万带 50 关注。
2. **必须打定位**。不打 = 主动放弃同城流量入口（同城占比的具体百分比无官方出处，但"快手同城权重远高于抖音"是多份券商研报共识）。
3. **评论必回 + 带 @ 粉丝名**。快手算法给"社区活跃账号"额外加权。
4. **人设 > 内容**。老铁关注的是"你这个人"，不是"你的内容"。
5. **不要照搬抖音逻辑**。抖音那套"高密度信息 + 快剪 + 字幕轰炸"在快手会扑街 — 快手要"慢一点、亲一点、土一点"。
6. **方言 / 地域元素是优势不是劣势**。普通话标准 + 精致包装在快手反而扣分。
7. **直播是必修课**，不是选修。快手主理人不直播 ≈ 抖音不发短视频。
